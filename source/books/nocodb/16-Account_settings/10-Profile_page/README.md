# 个人资料页面

个人资料页面是您管理个人资料信息的地方。目前，您可以在个人资料配置中设置自定义用户名。要访问您的个人资料页面，请执行以下步骤：

1. 点击侧边栏左下角的 `用户菜单`
2. 从下拉菜单中选择 `账户设置`
3. 修改 `个人资料名称`
4. 点击 `保存` 按钮以保存更改
