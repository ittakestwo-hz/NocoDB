# 搜索

NocoDB 提供了一个快速搜索功能，可以让您通过字段从工具栏菜单中搜索记录。这个功能在处理大数据集时非常有用，可以与 `Filter` 功能结合使用，以进一步细化搜索结果。

1. 点击工具栏菜单中的 `Search` 按钮（可选）
2. 选择要搜索的 `Field`（可选）
3. 在搜索栏中输入搜索词。
