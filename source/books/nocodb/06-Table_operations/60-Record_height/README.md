# 调整行高

NocoDB 为用户提供了在电子表网格中调整记录显示高度的灵活性，分为四个不同的级别：`Short`（默认设置）、`Medium`、`Tall` 和 `Extra`。该功能在处理长文本字段和多选字段时非常有用，允许用户在每个单元格中展示更多的内容。
