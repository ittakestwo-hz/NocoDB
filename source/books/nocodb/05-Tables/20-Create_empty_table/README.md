# 创建空表

## 创建新表

1. 点击左侧边栏 Base 名称旁边的 "+"，或在 Base 仪表盘上点击“创建新表”按钮。
2. 在弹出的对话框中填写表名（可选）。
3. 点击“添加描述”以给表添加描述（可选）。
4. 点击“创建表”按钮。
