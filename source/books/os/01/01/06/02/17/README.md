# 选择要安装的软件包

下一步允许你非常广泛地选择机器的用途；这十二个建议的任务对应着将要安装的软件包列表。实际安装的软件包列表将在后续步骤中进一步调整和完善，但这为你提供了一个简单的起点。

此步骤可能需要完整的安装媒体集，或者如上所述，要求有互联网连接和已配置好的镜像源。

根据检测到的硬件，一些软件包也会被自动安装（感谢 `discover` 包中的 `discover-pkginstall` 程序）。
