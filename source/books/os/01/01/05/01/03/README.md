# 与其他 Linux/Unix 系统的集成

最后，Debian 包括的 NFS（网络文件系统）和 NIS（网络信息服务）确保与 Unix 系统的互操作性。NFS 提供文件服务器功能，而 NIS 用于创建用户目录。大多数 Unix 系统使用的 BSD 打印层也允许共享打印队列。
